# 泡沫排序 (Bubble Sort)

使用 Python 實作的泡沫排序演算法。

## 什麼是泡沫排序？

泡沫排序是一種簡單的排序演算法，它通過重複比較相鄰的兩個元素，如果它們的順序錯誤就將它們交換。這個過程會一直重複，直到沒有任何交換需要進行為止。

## 時間複雜度

- **最壞情況：** O(n²)
- **最佳情況：** O(n) - 當陣列已經排序好時
- **平均情況：** O(n²)

## 使用方法

```bash
python bubble_sort.py
```

## 輸出範例

```
Original array: [64, 34, 25, 12, 22, 11, 90]
Sorted array: [11, 12, 22, 25, 34, 64, 90]
```
